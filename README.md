# Shockfall

Shockfall is an educational Unity project inspired by arcade-style arena survival games focused on physics-based gameplay, destructible environments, and stylized visuals.

The project was created as a learning experience for:
- Unity Gameplay Programming
- Physics Systems
- Lighting and Visual Direction
- Performance Optimization
- Level Design and Game Feel

## Project Information

- **Engine:** Unity 2022.3.62f3 LTS
- **Render Pipeline:** URP
- **Platform Target:** PC
- **Language:** C#

## Features

- Physics-based gameplay
- Arena survival mechanics
- Falling explosive hazards
- Stylized environment
- Optimization-focused workflow
- Modular project structure

## Goals

This project aims to improve technical and creative skills related to:
- Unity Development
- Gameplay Programming
- Real-time Lighting
- Game Systems Architecture
- Performance Optimization

## Status

Currently in active development as an educational and portfolio project.
